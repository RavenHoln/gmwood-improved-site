# Инструкция по размещению на GitHub Pages

## Шаги для активации GitHub Pages:

1. Перейдите в ваш репозиторий на GitHub: https://github.com/RavenHoln/gmwood-improved-site

2. Перейдите в настройки репозитория:
   - Нажмите на вкладку "Settings" (Настройки)

3. В левом меню найдите раздел "Pages" (Страницы)

4. В разделе "Source" (Источник) выберите:
   - Branch: `main`
   - Folder: `/ (root)`
   - Нажмите "Save" (Сохранить)

5. Через несколько минут ваш сайт будет доступен по адресу:
   `https://ravenholn.github.io/gmwood-improved-site/`

## Альтернативные варианты:

### Vercel (рекомендуется)
1. Зайдите на https://vercel.com
2. Войдите через GitHub
3. Нажмите "New Project"
4. Импортируйте репозиторий `gmwood-improved-site`
5. Нажмите "Deploy" - сайт будет автоматически размещен

### Cloudflare Pages
1. Зайдите на https://pages.cloudflare.com
2. Войдите через GitHub
3. Выберите репозиторий `gmwood-improved-site`
4. Настройки: Build command - оставьте пустым, Build output directory - `/`
5. Нажмите "Save and Deploy"

### Render
1. Зайдите на https://render.com
2. Войдите через GitHub
3. Создайте новый "Static Site"
4. Подключите репозиторий
5. Нажмите "Create Static Site"

Все эти сервисы бесплатные для статических сайтов и имеют хорошие лимиты.
