# Настройка Vercel (рекомендуется)

Vercel работает с приватными репозиториями БЕСПЛАТНО!

## Шаги:

1. Зайдите на сайт: https://vercel.com

2. Нажмите "Sign Up" (Регистрация) или "Log In" (Вход)

3. Выберите "Continue with GitHub" (Войти через GitHub)

4. Авторизуйтесь через GitHub

5. После входа нажмите кнопку "Add New..." → "Project" (Добавить проект)

6. Найдите в списке репозиторий: `gmwood-improved-site`

7. Нажмите "Import" (Импортировать)

8. В настройках проекта:
   - Framework Preset: оставьте "Other" или "Other (Static HTML)"
   - Root Directory: оставьте `./` (по умолчанию)
   - Build Command: оставьте пустым
   - Output Directory: оставьте пустым

9. Нажмите кнопку "Deploy" (Развернуть)

10. Через 1-2 минуты сайт будет готов!

11. Vercel автоматически даст вам адрес типа:
    `https://gmwood-improved-site.vercel.app`
    
    Или можно настроить свой домен.
